iPhone

Please click the link below to download the app from csc

FirstDoc Mobile 1.0 Application for LS Support
FirstDoc Mobile 1.0 - Eval environment
  1. After clicking the install button the app will install in the background. Go to home screen to launch the app.
  2. If you get the error that app is from untrusted developer then:
    In iOS 9.1 and lower, go to Settings → General → Profiles → tap on Profile ìComputer Science Corporationî → tap on Trust button.
    In iOS 9.2+, go to Settings → General → Device Management → tap on Profile ìComputer Science Corporationî → tap on Trust button.
